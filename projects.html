projects

HowStuffWorks Now

This is a social media driven content site for HowStuffWorks, in order to provide timely articles for current events. I have worked throughout the tech stack for this from the front-end SPA to our backend API services. It's core is built around AngularJS.

MakeStuff

MakeStuff is a full-service content management system that is designed to enable users to quickly generate content and increase social interaction. Currently, the content types that are supported are articles, quizzes, polls, and image galleries. The technologies behind this are AngularJS and LESS for the front-end components and Golang for the backend services.

Civic

Civic is a web-based CMS for local government and small cities. It enables users to quickly update residents with news, events, etc. It utilizes AngulasJS and Firebase.

Civic - Virginia City

This is an app I built for the town of Virginia City, Nevada. It is available on iTunes and Google Play. It is built off of the Civic framework, with Ionic. You should totally check it out.

Property Manager

This is an app I designed and built for use a local property management company, in order to streamline the accounting measures currently in place there. It logs account transactions, as well as manages rental and employee information. It uses Ruby on Rails and PostgresSQL.

SNPy: Genomics

Rails application that analyzes a 23andMe sequenced genome for known risk-factors. Utilizes an algorithm to accurately analyze and pinpoint risk factors by comparing the user's genome (uploaded as a txt file) with a database of known key-value pair risk-factors. Then displays a self-explanatory data visualization outlining the users risk-factors and their severity using JavaScript D3.
